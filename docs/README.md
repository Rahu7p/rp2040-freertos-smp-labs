# 📘 RP2040 FreeRTOS SMP Laboratory Manual

Welcome to the laboratory manual for **Multicore Embedded Systems using the RP2040 and FreeRTOS SMP**.

This manual has been designed for undergraduate Embedded Systems courses and guides the student from the development environment setup to the implementation and evaluation of multicore applications.

---

# Learning Objectives

After completing this manual, you will be able to:

- Configure a FreeRTOS SMP project for the RP2040.
- Understand the architecture of a multicore microcontroller.
- Design concurrent applications using both processor cores.
- Apply task affinity to improve system performance.
- Synchronize tasks using FreeRTOS mechanisms.
- Evaluate multicore performance through experimentation.
